# first-principle

`first-principle` is an interactive thinking-training skill based on Li Shanyou's *第一性原理*. It helps users strip away inherited assumptions, identify the deeper premise beneath a system, rebuild from a stronger first principle, and turn the result into a falsifiable action plan.

Use it when the user wants a structural rethink rather than incremental optimization.

## Best For

- Reframing a strategy, product, career, learning, or life problem from its base assumptions.
- Finding the hidden premise behind a current plan, business model, habit, or consensus.
- Breaking comparison thinking such as "others do it this way, so we should too."
- Moving from copied experience to deductive reasoning.
- Creating a non-incremental solution or new system from a clearer "one".
- Testing whether a proposed principle is logically and practically usable.

## Workflow

1. **Intake**: collect the user's problem, background, current approach, and constraints.
2. **System framing**: define what system is being examined and where its boundary may be breaking.
3. **Current logic**: express the inherited approach as `Because A, we do B, expecting C`.
4. **Assumption discovery**: ask 3 to 5 targeted questions to expose hidden assumptions.
5. **Rebuild**: propose candidate first principles and select one as the new base.
6. **Deduction**: derive a new model, strategy, behavior, or experiment from the selected principle.
7. **Validation**: check self-consistency, cross-system consistency, continuity, and falsifiability.
8. **Action plan**: produce a small test with success and failure signals.

## Example Invocation

```text
Use $first-principle to rethink our product strategy. We have been copying competitors' feature roadmap, but growth is flat. Background: ...
```

```text
使用 $first-principle 帮我用第一性原理重构这个问题：我现在的职业发展卡住了，总是在复制别人的路径。背景是……
```

## Notes

This skill is not a generic brainstorming tool. It should delay solution advice until the current system's hidden assumptions have been surfaced and a new base premise has been selected.
